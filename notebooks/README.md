# notebooks/

This folder contains all the Jupyter Notebooks I used during the model development process for the Retail Demand Forecast & Inventory Assistant.

## Purpose

These notebooks document the systematic experimentation and decision-making process behind the final forecasting model, including:

- Understanding the characteristics and limitations of the raw data
- Testing different feature engineering approaches
- Comparing multiple machine learning models
- Developing and tuning the final ensemble model
- Testing rolling forecast behaviour in a realistic setting

## Notebook List

| Notebook                                    | Description |
|---------------------------------------------|-----------|
| `Model_Exploration.ipynb`   | Notebook containing the complete modelling process |

## How to Use

You can open any notebook using Jupyter Notebook or JupyterLab.

These notebooks are intended for academic review and to demonstrate the development journey rather than for end-user execution.

---

**Note**:  
All major design decisions in the final `dashboard.py` (feature selection, ensemble weights, forecast horizon, etc.) were based on the experiments documented here.
